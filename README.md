# 人生重开模拟器

这是一个基于Next.js和React开发的人生重开模拟器应用，允许用户模拟一个全新的人生。

## 功能特点

- 属性系统：智力、外貌、财富、健康和幸运五大基本属性
- 天赋系统：选择不同天赋影响人生轨迹
- 事件系统：根据年龄和属性触发随机人生事件
- 选择系统：在关键时刻做出决策，影响后续发展
- 人生总结：展示你的人生成就和结局

## 技术栈

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Shadcn UI组件库

## 安装与运行

确保已安装Node.js环境，然后执行以下命令：

```bash
# 安装依赖
npm install

# 开发模式运行
npm run dev

# 构建项目
npm run build

# 运行生产环境
npm run start
```

## 游戏玩法

1. 在游戏开始前，调整各项属性上限和分配初始属性点
2. 选择适合的天赋以获得特殊加成
3. 开始模拟人生，每年会发生不同的事件
4. 遇到选择时做出决策
5. 人生结束后查看总结

## 自定义

可以通过修改代码轻松添加新的事件、天赋或游戏机制。
